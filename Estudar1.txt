# Recursos de Treinamento e Documentação

## Conceitos Importantes Independente de Plataforma
- **HTTP/HTTPS:**
  - Funcionamento, Principais cabeçalhos.

- **Balanceamento HTTP:**
  - Sessão, Afinidade de Sessão, Persistência de Sessão, Compartilhamento de Sessão.

- **Cache HTTP:**
  - Certificação Digital.

- **Monitoramento:**
  - Zabbix.
  - Prometheus.
  - Log / ELKG.
    - Elastic Search / Opensearch.
    - Logstash.
    - Filebeat.
  - Grafana.

- **Controle de versão:**
  - Git.

- **Backup:**
  - IBM Tivoli Storage Manager (TSM, também conhecido como IBM Spectrum Protect).

- **Virtualização:**
  - VMWare.

## Plataforma Linux
- **Sistema operacional:**
  - Ubuntu 22.04.

- **Orquestração de Servidores / Gerência de Configuração:**
  - Ansible.

- **Linguagens Comumente Utilizadas Para Administração:**
  - Bash.
  - Python.
  - Java (sistemas de automação de atividades do SESIA feitos pelo Robson Paniago).
  - Go templates (golang, pouquíssima coisa, apenas configuração do confd).

- **Plataformas de hospedagem de containers:**
  - Rancher 1.6, com cattle como orquestrador.
  - Rancher 2.6, com Kubernetes como orquestrador.
    - Obs: Praticamente todas as aplicações estão hospedadas no Rancher 1.6. Apenas os proxies reversos e algumas aplicações de uso exclusivo do SESIA estão no Rancher 2.

- **Proxy Reverso:**
  - Keepalived (VIP VRRP).
  - Nginx (terminador HTTPS, ponto de entrada HTTP, servidor de arquivos estáticos em alguns casos).
  - Varnish (cache HTTP, reescritas HTTP, WAF básico antigo adaptado o projeto OWASP).
  - Haproxy (balanceador, redirecionamentos HTTP).
  - Ingress HAProxy (balanceador).
  - Letsencrypt (gerador de certificados digitais).
  - Confd (automação da configuração do haproxy no ambiente Rancher 1.6).

## Plataforma Windows
- **Sistema operacional:**
  - Windows Server 2019.
  - Windows Server 2012 R2.

- **Proxy Reverso:**
  - ARR.

- **Balanceamento:**
  - Fortigate.

- **Servidores de aplicação:**
  - IIS.
  - OAS.

- **Linguagens Comumente Utilizadas Para Administração:**
  - Powershell.
